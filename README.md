## 🌈 灵动情感引擎：看见硬件的“呼吸”

Comfypet 拥有独特的**硬件感知系统**，它会根据您物理机器的“心跳”实时切换视觉状态：

- **🧊 静谧蓝 (Calm)**: 显存占用低于 40%，桌宠处于待机或轻量浏览模式，安稳陪伴。
- **⚡ 脉冲黄 (Thinking)**: ComfyUI 正在运行或显存由于生成任务升高，桌宠进入“思考/任务”状态。
- **🔥 警戒红 (Critical)**: 显存接近溢出（>90%）或 GPU 满载，桌宠会表现出“头晕/过载”反馈，甚至流汗提醒。

## ⏰ 24 小时全天候交互

桌宠不仅仅是监控器，它还具备**时间感知能力**：

- **🌅 清晨 (06-11)**: 桌宠精神抖擞，会有更多的随机动作和“闪烁”神情。
- **🛋️ 午后 (11-18)**: 专注稳定的工作状态，伴随您的生产力稳健运行。
- **🌙 深夜 (23-06)**: 自动进入**深度睡眠模式**。窗口会变暗，桌宠会闭上眼睛打鼾，最大程度减少对您深夜工作的视觉干扰。

## ✨ 核心特性

### 1. ComfyUI 多实例管理
- **一键启停**：支持维护多个启动配置，一键批量开启/关闭多个 ComfyUI 实例。
- **环境隔离**：通过配置不同的 Python 路径和工作目录，轻松管理复杂环境。

### 2. Workflow Center (工作流中心)
- **资产直管**：一键导入本地 JSON 工作流，自定义参数暴露。
- **智能桥接 (Agent Bridge)**：内置 HTTP 服务，支持通过 API 远程触发工作流，完美对接 AI Agent。

### 3. 硬件 HUD 监控
- **实时监控**：透明窗体展示内存、NVIDIA GPU 显存占用，压力实时反馈。

## 📖 安装向导

### Windows 用户
1. 下载 `JMCAI-1.2.7-setup.exe`。
2. 运行并完成安装，桌面将出现 `JMCAI` 图标。

### macOS 用户
1. 下载 `JMCAI-1.2.7.dmg`。
2. 将 `JMCAI` 拖入 `Applications` 文件夹。
3. **注意**：若提示“无法验证开发者”，请在图标上**点击右键 -> 选择“打开”**。
