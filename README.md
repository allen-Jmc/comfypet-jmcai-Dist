## ✨ 核心特性

### 1. 灵动的桌宠交互 (Mascot Core)
- **情绪反馈**：桌宠会根据您的硬件压力（GPU/内存）和 AI 任务状态实时切换表情与动作。
- **轻量入口**：常驻桌面，通过简单的点击即可唤起控制台、监控中心。

### 2. ComfyUI 多实例管理
- **一键启停**：支持维护多个启动配置，一键批量开启/关闭多个 ComfyUI 实例。
- **状态监控**：实时查看端口、PID 以及运行日志。
- **环境隔离**：通过配置不同的 Python 路径和工作目录，轻松管理复杂环境。

### 3. Workflow Center (工作流中心)
- **资产直管**：导入本地 JSON 工作流，自定义参数暴露。
- **智能桥接 (Agent Bridge)**：内置 HTTP 服务，支持通过 API 远程触发工作流，完美对接 AI Agent。
- **运行记录**：查看历史生成的图片、视频及执行日志。

### 4. 硬件 HUD 监控
- **实时监控**：透明窗体展示内存、NVIDIA GPU 显存占用。
- **压力感知**：当 AI 系统高负荷运转时，桌宠会进入对应的“忙碌”或“兴奋”状态。

### 5. 跨平台支持
- **Windows**：深度优化，支持托盘常驻与系统联动。
- **macOS**：原生支持，界面采用高通透玻璃拟态设计（Fluid Glass UI）。

## 📖 安装向导

### Windows 用户
1. 下载 `JMCAI-1.2.0-setup.exe`。
2. 运行并完成安装，桌面将出现 `JMCAI` 图标。
3. 首次启动后，在“设置”中配置您的 Python 路径和 ComfyUI 工作目录。

### macOS 用户
1. 下载 `JMCAI-1.2.0.dmg`。
2. 将 `JMCAI` 拖入 `Applications` 文件夹。
3. **注意**：若提示“无法验证开发者”，请在应用图标上**点击右键 -> 选择“打开”**，并在弹出框中再次确认。
